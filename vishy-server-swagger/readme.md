vishy-server-swagger
=======



e.g. in your `application.yml`:

  description: The OpenMRC REST API interface definition;
  termsOfServiceUrl: ${TERMS_OF_SERVICE_URL};
  license: MIT License;
  licenseUrl: https://opensource.org/licenses/MIT;
  contact: ${AUTHOR};
